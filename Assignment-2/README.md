This repo includes programs demonstrating:
- Syntax and semantics in Python, JavaScript, and C++
- Closures, dynamic typing, static typing, and coercion
- Memory management in Rust, Java, and C++
- Ownership and borrowing (Rust)
- Garbage collection (Java)
- Manual memory management (C++)

---

## File Extensions Used
CPP → `.cpp`  
RUST → `.rs`  
JS → `.js`  
JAVA → `.java`  
PYTHON → `.py`  

---

## How to Run All Files with different extensions

1. C++ (.cpp)
   - Compile: g++ filename.cpp -o output
   - Run: ./output
   - Windows: output.exe

2. Rust (.rs)
   - Compile: rustc filename.rs
   - Run: ./filename
   - Windows: filename.exe

3. JavaScript (.js)
   - Run: node filename.js

4. Java (.java)
   - Compile: javac Filename.java
   - Run: java Filename

5. Python (.py)
   - Run: python filename.py
